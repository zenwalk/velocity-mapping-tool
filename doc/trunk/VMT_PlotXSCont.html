VMT_PlotXSCont

PURPOSE

Plots contours for the variable 'var' within the mean cross section given

SYNOPSIS

function [z,A,V,zmin,zmax,log_text,fig_contour_handle] = VMT_PlotXSCont(z,A,V,var,exag,plot_english)

DESCRIPTION 

 Plots contours for the variable 'var' within the mean cross section given
